# sales-kpi-dashboard-sql-powerbi
Built an end-to-end Sales KPI Dashboard using SQL and Power BI, analyzing 10,000+ records to track revenue trends, regional performance, and data quality metrics.
Objective
Analyze sales data and build an interactive dashboard to track revenue, profit, and regional performance.
Tools Used
SQL
Power BI
Excel
Key Insights
Central region generated highest sales
OfficeSuppliers category contributed most revenue
High discounts reduced profit margins
Sales show seasonal trends
Features
KPI Cards (Sales, Profit, Quantity,Average Delivery Days)
Region-wise Analysis
Category-wise Performance
Time-based Trends
Data Quality Checks
Identified null values
Removed duplicates
Checked data consistency
